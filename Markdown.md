### MarkDown

- ".md" 확장자를 가지고 있음.
- 매우 간단한 문법과 다양한 기능이 있음.

<br/>

### 제목

- #. 가장 큰 제목
- ##. 두번째로 큰 제목
- ###. 세번째로 큰 제목
- ####. 세번째로 작은 제목
- #####. 두번째로 작은 제목
- ######. 가장 작은 제목

<br/>

### 텍스트 포맷팅

- _이탤릭체_ (\*)
- **강조체** (\*\*)
- ~~취소선~~ (~~)
- 'int age = 22'
- 줄바꿈 (br/)
- (u) 밑줄 (u/)
- 작업 목록 ([] or [x])
- > ">"
- > > ">>"

<br/>

### 아이템과 순서

- 1
  - 2
    - 3
  - 2
- 1

### 링크와 사진

- 링크 삽입
  - [Site_Name] ( URL )

<br/>

- 사진 삽입
